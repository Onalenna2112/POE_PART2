# CyberBot – Cybersecurity Awareness Chatbot

## Project Overview

CyberBot is a cybersecurity awareness chatbot developed using C# Windows Forms in Visual Studio.

The purpose of the application is to educate users about cybersecurity topics such as password safety, phishing, scams, privacy, malware, VPNs, and safe browsing.

The chatbot includes:
- Keyword recognition
- Random responses
- Sentiment detection
- Memory and recall
- Cybersecurity quiz
- Chat history
- ASCII art
- Audio greeting
- GUI design

---

## How the Program Works

When the application starts, a greeting sound plays and the graphical user interface opens.

Users interact with the chatbot by typing cybersecurity-related keywords or questions. The chatbot detects keywords and responds with relevant cybersecurity advice.

The chatbot can also:
- Detect emotions such as worried or confused
- Remember previous topics discussed
- Generate random cybersecurity tips
- Save chat history
- Provide a cybersecurity quiz

## How to Run the Program

Requirements:
- Visual Studio
- .NET Framework

Steps:
1. Download or clone the repository.
2. Open the solution in Visual Studio.
3. Build the project using:
   Ctrl + Shift + B
4. Run the application using:
   F5
