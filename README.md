# CyberBot – Cybersecurity Awareness Chatbot

## Project Overview

CyberBot is a cybersecurity awareness chatbot developed using C# Windows Forms in Visual Studio.

The purpose of the application is to educate users about cybersecurity topics such as password safety, phishing, scams, privacy, malware, VPNs, and safe browsing.

The chatbot includes:
- Keyword recognition
- Random responses
- Sentiment detection
- Memory and recall
- Cybersecurity quiz
- Chat history
- ASCII art
- Audio greeting
- GUI design

---

## How the Program Works

When the application starts, a greeting sound plays and the graphical user interface opens.

Users interact with the chatbot by typing cybersecurity-related keywords or questions. The chatbot detects keywords and responds with relevant cybersecurity advice.

The chatbot can also:
- Detect emotions such as worried or confused
- Remember previous topics discussed
- Generate random cybersecurity tips
- Save chat history
- Provide a cybersecurity quiz

## 📖 Project Description

The Cybersecurity Awareness Chatbot is a Windows Forms application developed in C# to educate users about cybersecurity best practices. The chatbot provides information on topics such as passwords, phishing, scams, malware, privacy, and VPNs.

The application also includes:

- A Task Assistant for cybersecurity-related tasks.
- Reminder functionality with custom dates or timeframes.
- MySQL database integration.
- A cybersecurity mini-game/quiz.
- NLP simulation using keyword detection and string manipulation.
- Activity logging for user interactions and chatbot actions.
- Memory management and sentiment analysis.

This project combines the functionality developed across Parts 1, 2, 3, and 4 of the Portfolio of Evidence (POE).
