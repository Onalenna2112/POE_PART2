# CyberBot – Cybersecurity Awareness Chatbot

## Project Overview

CyberBot is a cybersecurity awareness chatbot developed using C# Windows Forms in Visual Studio.

The chatbot helps users learn about cybersecurity topics such as:

* Password Safety
* Phishing
* Online Scams
* Privacy
* Malware
* VPNs

The application includes:

* Keyword recognition
* Random responses
* Memory recall
* Sentiment detection
* Quiz functionality
* Chat history
* ASCII art
* Audio greeting
* Professional GUI design
